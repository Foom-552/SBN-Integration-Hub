# SBN Integration Simulation Hub

An interactive training and demonstration tool for the **SAP Business Network** supplier integration process.

This single-page web application walks users through the complete cXML integration lifecycle, including:

- **Overview** of SAP Business Network and integration models
- **7-Phase Process** timeline for supplier onboarding
- **Document Standards** reference for cXML, X12, EDIFACT, EANCOM, and PIDX
- **Managed Gateway** setup wizard
- **GITP** (Global Integration Trading Partner) guided walkthrough
- **Test Central** scenario builder for UAT planning
- **UAT Script Builder** for tracking test cases
- **Timeline Planner** with holiday-aware scheduling for 14 countries
